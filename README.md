## 搬瓦工升级套餐是什么意思？

升级套餐，简单说就是：把你现在在用的 VPS 换成配置更高或线路更好的方案，并且只需要补交差价，不用重新买一台机器，原来的数据和 IP 也不受影响（如果升到同机房更高配置的话）。

搬瓦工支持的升级方向主要有两种：

1. **同系列升高配**：比如 KVM 1TB 流量升到 2TB，配置变大，IP 和机房不变
2. **跨系列升级**：比如从 KVM 或 CN2 套餐升级到 CN2 GIA-E，线路档次提升，但机房和 IP 会变

这两种情况有个重要区别：**跨系列升级会更换 IP**，如果你有业务绑定在当前 IP 上，升级前要提前做好准备。

## 搬瓦工升级套餐操作步骤

操作本身不复杂，后台点几下就完成了。

1. **登录后台**：进入搬瓦工用户中心，点击右上角 Client Area
2. **找到当前服务**：依次点击 Services → My Services，找到要升级的 VPS 实例
3. **进入管理页**：点击对应实例的 Manage 按钮
4. **发起升级**：在 Management Actions 里点 Upgrade/Downgrade
5. **选目标方案**：后台会列出可以升级的目标套餐，每个旁边有一个 "Upgrade to this plan" 按钮
6. **确认差价并付款**：系统会自动计算当前剩余时长对应的差价，确认金额后付款即可

整个过程两三分钟能搞定。付款成功后套餐立刻生效，不需要等待。

## 升级差价怎么算？

不少人纠结这个问题，其实逻辑很直白。

公式是：**差价 = (目标套餐年价 - 当前套餐已支付价格) × 剩余天数/总天数**

如果你购买时用了优惠码，计算过程中优惠码折扣会同步体现。

举个例子：假设你当初花了 $46.7（用优惠码后）买了一年 KVM 套餐，用了 90 天后想升级到 CN2 GIA-E（年付 $169.99），剩余 275 天，那差价大概是：

($169.99 - $46.7) × (275/365) ≈ **$92.88**

付了这个差价，你的套餐就从 KVM 变成 CN2 GIA-E，有效期不变，从剩余的 275 天继续算。

还有一点：**升级套餐目前不支持降级**，也就是说升上去之后不能再自助降回来。如果确实需要降级，要提交工单联系客服处理，但实际很麻烦，能否降也不一定。所以升级前想清楚，别仅仅因为"想试试"就贸然操作。

## 哪种情况值得升级？哪种情况不用急

**值得升级的情况：**

- 流量月月告急，感觉每次都在捏着用
- 晚高峰经常卡，明显感觉到线路质量跟不上
- 从 KVM 用起，已经稳定用了一段时间，想换到 CN2 GIA-E 试试质量差距

**不一定要升级的情况：**

- 手里有限量版"传家宝"套餐（比如 THE PLAN、BiggerBox Pro 这类），**强烈建议不要轻易升级**。这类套餐是特定时期补货抢到的，配置好、价格低，升级后大概率拿不回来
- 流量偶尔超一点，但不是每个月都超——可以先观察几个月再决定
- 当前套餐剩余时间很短（比如不到两个月），差价划不来，不如等到期再买新套餐

说实话，升级这个功能最大的意义，是给那些流量真的用完、或者线路已经明确不够用的人提供一个平滑过渡通道，不是让你为了升级而升级。

## 常见问题

**Q：升级后原来的 IP 还在吗？**  
A：如果是升级到同机房的更高配套餐，IP 不变。如果是跨系列升级（比如 KVM 升到 CN2 GIA-E），机房和 IP 都会更换，升级前需要做好迁移准备。

**Q：升级时能不能选机房？**  
A：升级操作会把你迁入目标套餐对应的机房体系，迁到哪个具体机房，通常在升级后可以在 KiwiVM 后台自行切换。

**Q：升级之后还能再次升级吗？**  
A：可以。比如 CN2 GIA-E 1TB 流量版，后续还可以升级到 2TB 流量版，操作方式相同，补差价。

**Q：当前有没有升级套餐的优惠码？**  
A：目前暂无可用的常规优惠码。搬瓦工一年通常只有双十一和黑五两个固定促销节点，建议到时候留意官网。如果着急用，就按原价升级，不用等。

**Q：降级套餐怎么操作？**  
A：目前不支持自助降级，需要提交工单联系官方客服处理。日常使用建议不要随意升级高配，避免后续有降级需求。
